📚 Data Note: Publication Year

The pub_year in this dataset comes from Goodreads. A sample of the 400 most-rated books was cross-checked against the Open Library API (which returns first-publish dates from library records). No systematic discrepancies were found — the Goodreads publication year is the actual book publication date, not the review posting date. The 2016 peak and post-2017 drop visible in the trend chart is a real phenomenon, not a data artifact.

Source: 400 Open Library queries via /search.json?q=title+author, validated with word-overlap matching (threshold ≥50%).
